# Кейсы

## Главный тезис

У `Мастерской лидов` есть практический опыт в нескольких нишах, особенно в строительстве, ремонте, недвижимости и экспертных услугах.

## Ключевые кейсы

### 1. Альянс Строителей Крыма

- ниша: строительство домов;
- регион: Крым;
- срок работы: `4 года`;
- результат: `18 000 подписчиков` в сообщество;
- коммерческий итог: выручка через ВК превысила `100 млн ₽`.

Как презентовать:

> Один из самых сильных долгосрочных кейсов компании. За несколько лет через ВКонтакте удалось не просто раскачать сообщество, а довести результат до выручки свыше 100 млн рублей.

### 2. КапиталСтрой

- ниша: ремонт квартир;
- регион: Тамбов;
- срок работы: `2.5 года`;
- результат: более `7 000 подписчиков`;
- продажи: `62 договора`;
- средний чек договора: `1.1 млн ₽`.

Как презентовать:

> Это кейс, который хорошо показывает связь между таргетированной рекламой ВК и реальными продажами: не просто лиды, а десятки закрытых договоров.

### 3. FreeDom

- ниша: строительство домов;
- регион: Кострома;
- срок работы: `5-й месяц`;
- результат по подписчикам: `+1064` по `46 ₽`;
- результат по заявкам: `132 заявки` в среднем по `800 ₽`.

Как презентовать:

> Хороший пример активного текущего проекта, где одновременно виден и рост сообщества, и стабильный поток заявок.

### 4. ЕвроДом

- ниша: ремонт квартир;
- регион: Смоленск;
- срок работы: `4 месяца`;
- результат по подписчикам: `+1190` по `21 ₽`;
- результат по заявкам: `67 заявок` по `1417 ₽` в среднем.

Промежуточная воронка по состоянию на `8 февраля`:

- бюджет: `148 730 ₽`;
- лиды: `71`;
- квалифицированные лиды: `44`;
- замеры: `20`;
- договоры: `7`;
- сумма по договорам: `6.3 млн ₽`.

Как презентовать:

> Сильный кейс для клиентов из ремонта: можно показать не только цену лида, но и движение до договоров и итоговой выручки.

### 5. Пластический хирург

- ниша: пластическая хирургия;
- регион: Москва;
- срок работы: `2-й месяц`;
- результат по подписчикам: `+714` по `23 ₽`;
- результат по заявкам: `41 заявка` по `1018 ₽`.

Как презентовать:

> Этот кейс особенно полезен для медицинских и экспертных ниш, где важны доверие, аккуратная подача и качество аудитории.

## Дополнительные тезисы для продаж

- это только часть проектов компании;
- больше кейсов опубликовано на стене сообщества, в статьях и в отдельном разделе с кейсами;
- при разговоре лучше подбирать кейс максимально близкий к нише клиента;
- если полного совпадения по нише нет, показываем близкую механику: локальный бизнес, экспертная услуга, дорогой чек, длительный цикл сделки.
